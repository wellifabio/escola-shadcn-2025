# Escola - Gestão de atividades
## Tecnologias
- React
- TypeScript
- Vite
- Shadcn

## Passos para executar
1. acesse a pasta do front end `./web` instale as dependências e inicie o servidor de desenvolvimento
```bash
cd web
npm install
npm run dev
```
Clicar no link localhost para executar
